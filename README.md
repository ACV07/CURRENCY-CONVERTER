# CURRENCY-CONVERTER

Simple Java program to convert currencies (project contains `CurrencyConverter.java`).

## What this project is
A minimal example project showing a small currency converter written in Java. This repo contains the single source file `CurrencyConverter.java`.

## Requirements
- Java JDK 8 or newer installed and available on your PATH.

## How to compile and run (PowerShell)
```powershell
javac .\CurrencyConverter.java
java CurrencyConverter
```

If the program expects command-line arguments, pass them after `CurrencyConverter` (for example: `java CurrencyConverter 100 USD EUR`).
